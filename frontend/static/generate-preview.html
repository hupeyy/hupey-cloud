Model Preview Generator

Instructions:

  1. Select a model from the dropdown
  2. The preview will update automatically
  3. Right-click the preview and "Save image as..."
  4. Save it in your static/images/previews/ folder
  5. Name it to match your model (e.g., "hupey-cloud.png")

Alternative Method:

If you prefer to use your actual 3D models:

  1. Run npm run dev
  2. Go to http://localhost:5173/model-preview
  3. Use the browser-based generator there
